# Navigation

## Overview

The Omarchy system operates primarily through keyboard commands rather than mouse interaction. Everything in Omarchy happens via the keyboard.

## Launchers

- System launcher: `Super + Space`
- Omarchy Menu: `Super + Alt + Space`

## Core Navigation Commands

### Application Launching

| Action | Shortcut |
|--------|----------|
| Terminal | `Super + Return` |
| Browser | `Super + Shift + B` |
| Activity Monitor | `Super + Shift + T` |
| File Manager | `Super + Shift + F` |

### Window Management

| Action | Shortcut |
|--------|----------|
| Stack windows horizontally | `Super + J` |
| Switch focus between windows | `Super + Arrow` |
| Swap windows | `Super + Shift + Arrow Right` |
| Move applications between workspaces | `Super + Shift + [number]` |
| Close windows | `Super + W` |
| Toggle floating mode | `Super + T` |
| Full screen | `Super + F` |
| Full-width view | `Super + Alt + F` |

### Advanced Features

- **Rearrange windows with mouse**: Hold `Super` and click to move; hold `Super + right-click` to resize
- **Window grouping**: `Super + G` to create groups; `Super + Alt + Arrows` to navigate between grouped windows
- **Move window from group**: `Super + Alt + G`
- **Scratchpad workspace (overlay)**: `Super + S` to access; `Super + Alt + S` to place windows there
