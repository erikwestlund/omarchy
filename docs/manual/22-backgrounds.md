# Backgrounds

## Storage Location

Backgrounds are stored in `~/.config/omarchy/[theme]/backgrounds`

To add images to a specific theme like Nord, place files in:
```
~/.config/omarchy/nord/backgrounds
```

## Installation Method

Use the graphical approach via the Omarchy Menu:

1. Navigate to **Install > Style > Background**
2. This opens the backgrounds folder for your current theme
3. Press `Super + Shift + F` to open a second file manager window
4. Locate your desired background image
5. Copy it to the theme's backgrounds directory

## Cycling Backgrounds

Cycle through available backgrounds using:
```
Super + Ctrl + Space
```
