# GUIs (Graphical Applications)

## Free/Open Source Applications

### Obsidian
A free, extensible note-taking app using Markdown files.
- Launch: `Super + Shift + O`
- Optional commercial sync add-on available for mobile devices

### Pinta
A basic image editing tool that's great for cropping, resizing, and other basic manipulations.
- Launch: Application launcher (`Super + Space`)

### LocalSend
Cross-platform file sharing tool similar to AirDrop, compatible with Windows, macOS, Android, iOS, and Linux devices on the same network.

### LibreOffice
Complete office suite including word processor, spreadsheet, presentations, and drawing tools. Compatible with Microsoft Office formats.
- Launch: `Super + Space`

### Signal
The pioneer of E2E encrypted messaging for secure communications.
- Launch: `Super + Shift + G`

### mpv
Media player supporting diverse video sources and formats.
- Launch: Application launcher or double-click videos in file manager

### OBS Studio
Recording and streaming software supporting multiple inputs and mixing capabilities.
- Launch: `Super + Space`

### Kdenlive
Video editing application suitable for post-processing OBS Studio content.
- Launch: `Super + Space`

## Commercial Applications

Omarchy is mostly focused on providing free, open source software, but it's not religious about it.

### 1Password
Password management tool for teams. Includes command-line integration for scripts.
- Launch: `Super + Shift + /`

### Typora
Markdown-based writing application emphasizing distraction-free composition. Minimal interface similar to iA Writer.
- 15-day free trial, then $15 one-time cost
- Launch: `Super + Space`
- Fullscreen: F11

### Dropbox
Cloud file synchronization and backup service.
- Install: _Install > Service > Dropbox_ from Omarchy menu

### Tailscale
Mesh VPN for secure multi-device networking.
- Install: _Install > Service > Tailscale_ from Omarchy menu

### Spotify
Music streaming service with offline playback capability.
- Launch: `Super + Shift + M`
