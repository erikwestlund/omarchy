# Web Apps

## Installation & Management

Add custom web apps via "Install > Web App" in the Omarchy menu (`Super + Alt + Space`). The process requires:
- App name
- URL
- Icon URL (can be sourced from Dashboard Icons)

Apps launch through the application launcher (`Super + Space`).

Removal occurs through "Remove > Web App" in the same menu.

Copy web app URLs to clipboard using `Shift + Alt + L`.
