# TUIs (Terminal User Interfaces)

## Lazygit

A delightful alternative to something like the GitHub Desktop application, and it runs inside the terminal.

- Launch: `lazygit` in any git directory or `Space G G` within Neovim
- Navigate panes: `Tab`
- Stage files: `Space`
- Create commits: `c`
- View all commands: `?`

## Lazydocker

A terminal interface for managing your containers and images.

- Launch: `Super + Shift + D`
- Stop containers: `s`
- Restart containers: `r`
- View commands: `?`

## Btop

A beautiful resource manager that shows memory, CPU, disk, and network usage. It also lists all active processes, and allows you to manage them.

- Launch: `Super + Shift + T` or via Activity application

## Impala

Wi-Fi management interface accessed through the top menu bar's Wi-Fi icon.

- Navigate sections: `Tab`
- Select networks: `Space`
- Enter passwords when required

## BlueTUI

Handles Bluetooth connections using identical navigation mechanics to Impala.

## Fastfetch

System information display (successor to neofetch) showing kernel version, uptime, theme, CPU, and memory details.

- Access: _About_ in the Omarchy menu (`Super + Alt + Space`)
