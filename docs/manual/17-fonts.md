# Fonts

## Default Font

Omarchy uses **JetBrainsMono Nerd Font** as both the terminal and system font by default.

## Changing Fonts

Modify the terminal font through:
- Style > Font menu
- Accessible via the Omarchy menu (`Super + Alt + Space`)

## Installing Additional Fonts

Install popular programming fonts through:
- Install > Style > Font menu in the Omarchy menu

This allows users to expand font options beyond the default selection without manual installation.
