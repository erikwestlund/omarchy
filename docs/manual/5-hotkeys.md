# Hotkeys

You can see all the main keyboard bindings by hitting `Super + K`.

## Navigating

| Action | Shortcut |
|--------|----------|
| Jump to workspace 1/2/3/4 | `Super + 1/2/3/4` |
| Move to next workspace | `Super + Tab` |
| Move focus | `Super + Arrow keys` |
| Swap windows | `Super + Shift + Arrow` |
| Grow window horizontally | `Super + Equal` |
| Shrink window horizontally | `Super + Minus` |
| Grow window vertically | `Super + Shift + Equal` |
| Shrink window vertically | `Super + Shift + Minus` |
| Toggle window grouping | `Super + G` |
| Navigate grouped windows | `Super + Alt + Arrows` |
| Browser tab jump | `Ctrl + 1/2/3` |

## Launching Applications

| Application | Shortcut |
|-------------|----------|
| Terminal | `Super + Shift + Return` |
| Browser | `Super + Shift + B` |
| File manager | `Super + Shift + F` |
| Music (Spotify) | `Super + Shift + M` |
| Password manager (1Password) | `Super + Shift + /` |
| Calendar (HEY) | `Super + Shift + C` |
| Email (HEY) | `Super + Shift + E` |
| Messenger (Signal, WhatsApp, Google) | `Super + Shift + G` variants |

## Universal Clipboard

Usually on Linux, you need `Ctrl + Shift + C/V` to copy'n'paste in the terminal, but Omarchy provides unified shortcuts:

| Action | Shortcut |
|--------|----------|
| Copy | `Super + C` |
| Cut | `Super + X` |
| Paste | `Super + V` |
| Clipboard manager | `Super + Ctrl + V` |

## Capture Tools

| Action | Shortcut |
|--------|----------|
| Screenshot with editing | `Print Screen` |
| Screenshot to clipboard | `Shift + Print Screen` |
| Screen record | `Alt + Print Screen` |
| Color picker | `Super + Print Screen` |

## Notifications & Style

| Action | Shortcut |
|--------|----------|
| Theme selection | `Super + Ctrl + Shift + Space` |
| Background cycling | `Super + Ctrl + Space` |
| Notification management | `Super + comma` variants |

## Specialized Tools

### Ghostty Terminal
Split and tab management via `Ctrl + Shift` combinations.

### File Manager
- Path navigation: `Ctrl + L`
- File preview: `Space`

### Neovim
Space-based command system with sidebar and file navigation controls.

### Quick Emojis
`CapsLock + M` combinations for rapid emoji access.

### Quick Completions
`CapsLock + Space` for mdash, name, and email insertion.

## Custom Bindings

Custom bindings can be modified in `~/.config/hypr/bindings.conf`.
