# The Applications

Omarchy comes pre-configured with a curated selection of applications:

## Included Applications

- **Neovim** - Terminal-based text editor
- **Ghostty** - Terminal emulator
- **Chromium** - Web browser
- **Spotify** - Music streaming
- **Typora** - Markdown editor
- **LibreOffice** - Office suite
- **1Password** - Password manager
- **Signal** - Messaging
- **WhatsApp** - Messaging

## Application Shortcuts

| Application | Shortcut |
|-------------|----------|
| Terminal | `Super + Shift + Return` |
| Browser | `Super + Shift + B` |
| File manager | `Super + Shift + F` |
| Music (Spotify) | `Super + Shift + M` |
| Password manager (1Password) | `Super + Shift + /` |
| Calendar (HEY) | `Super + Shift + C` |
| Email (HEY) | `Super + Shift + E` |

## PDF Handling

### Document Viewer
Basic PDF viewer that opens when double-clicking PDFs. Only works with PDFs already formatted as fillable forms.

### Xournal++
For PDFs lacking form setup or those requiring signatures, right-click the file and select "Open With..." to access Xournal++:
- Text entry anywhere on document using the T tool
- Signature insertion via the Image tool (requires a signature image file)
- Image resizing functionality

After completing edits, export through the File menu to save the modified PDF.
